Change login password

The new password is stored in R2 as a salted SHA-256 hash at the path configured by PASSWORD_PATH in wrangler.toml. On a fresh deploy — when that file doesn't exist yet — the default login password is 123456; change it here to materialise the file. Changing it signs every other existing session out; you stay signed in on this browser.

Sign in required

You need to sign in as admin to change the password.
